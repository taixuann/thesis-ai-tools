# Document Structure Templates

## Standard IMRaD Paper

```
1. Title
2. Abstract (structured: Background / Methods / Results / Conclusion)
3. Introduction (funnel: broad → narrow → gap → contribution → outline)
4. Methods
   4.1 Materials / Sample Preparation
   4.2 Device Fabrication
   4.3 Measurement Setup
   4.4 Characterization
   4.5 Data Analysis
5. Results
6. Discussion
   6.1 Key Findings Summary
   6.2 Interpretation and Mechanism
   6.3 Comparison to Literature
   6.4 Limitations
7. Conclusion
8. Acknowledgments
9. References
```

## Extended Paper (IMRaD + Theory)

```
1. Title
2. Abstract
3. Introduction
4. Theory / Background
   4.1 Governing Equations
   4.2 Model Derivation
   4.3 Key Assumptions
5. Methods (experimental + computational)
6. Results
7. Discussion (or integrated "Results and Discussion")
8. Conclusion
9. Appendices (derivations, extended proofs)
10. References
```

## Thesis Chapter

```
Chapter N: [Title]
N.1 Introduction (chapter-level funnel)
    N.1.1 Motivation and Context
    N.1.2 Problem Statement
    N.1.3 Chapter Contributions
    N.1.4 Chapter Outline
N.2 Background / Theory (if not in dedicated background chapter)
N.3 Methods (chapter-specific)
N.4 Results
    N.4.1 [Experiment / Study 1]
    N.4.2 [Experiment / Study 2]
N.5 Discussion (chapter-level)
N.6 Conclusion
    N.6.1 Summary of Findings
    N.6.2 Contributions
    N.6.3 Outlook (link to next chapter)
```

## Review Article

```
1. Title
2. Abstract
3. Introduction (scope, motivation, organization)
4. Background / Fundamentals (for non-specialists)
5. Thematic Sections (by concept, chronologically, or by technique)
6. Synthesis and Outlook
   6.1 Cross-cutting Themes
   6.2 Open Questions
   6.3 Future Directions
7. Conclusions
8. References (extensive, curated)
```
