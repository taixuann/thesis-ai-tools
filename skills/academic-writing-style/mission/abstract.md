# Abstract — Structured Format

Load this module when drafting or reviewing an Abstract.

Write it *last*, but place it *first*.

## Structured Format

4–5 sentences minimum, one paragraph:

```
Background (1 sentence):    What is the broader context or problem?
Methods (1–2 sentences):    What did you do? (Key approach)
Results (1–2 sentences):    What did you find? (Key quantitative findings)
Conclusion (1 sentence):    What does it mean? (The nugget, implications)
```

## Rules

- **Self-contained**: No citations to the paper's own sections, no undefined abbreviations
- **Quantitative**: Include key numbers — "a 3× improvement", not "significant improvement"
- **No references** to other papers (exceptions: landmark method papers)
- **Past tense** for what you did and found; **present tense** for what findings mean
- **The nugget**: Final sentence must crystallize the single key insight
