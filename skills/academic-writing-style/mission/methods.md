# Methods — Chronological Recipe

Load this module when drafting or reviewing a Methods section.

## Voice and Tense

- **Past tense** throughout (you already did it)
- **Third-person passive** as the default: "The samples were annealed..."
- **Active voice** only when justifying a choice: "We selected RF sputtering over thermal evaporation because..."

## Structure

Follow the chronological order of the experiment:

1. **Materials / Sample preparation** — Substrate, deposition, patterning, annealing
2. **Device fabrication** — Layer stack, dimensions, electrode deposition
3. **Measurement setup** — Equipment, conditions (temperature, atmosphere, probe configuration)
4. **Characterization techniques** — XRD, XPS, TEM, AFM — with instrument models and settings
5. **Data analysis** — How raw data was processed, statistical methods, software used

## Instrumentation Specs

Must include: manufacturer, model, and key operating parameters.

> "XRD patterns were collected using a Rigaku SmartLab diffractometer (Cu Kα, λ = 1.5406 Å, 40 kV, 44 mA, 2θ range 20–80°, step size 0.02°)."
